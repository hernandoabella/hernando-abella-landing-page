Tutorial de Node.js

Introducción

¿Qué es Node.js?

  • Node.js es un entorno de servidor de código abierto
  • Node.js es gratuito
  • Node.js se ejecuta en varias plataformas (Windows, Linux, Unix, Mac OS X, ect.)
  • Node.js utiliza JavaScript en el servidor.

¿Por qué Node.js?

Nota: Node.js usa programación asincrónica

Una tarea común para un servior web puee ser abrir un archivo en el servidor y devolver el contenido al cliente.

Así es como PHP o ASP manejan una solicitud de archivo:

  1. Envía la tarea al sistema de archivos de la computadora.
  2. Espera mientras se abre el sistema de archivos y lee el archivo.
  3. Devuelve el contenido al cliente.
  4. Listo para manejar la próxima solicitud.

Así es como Node.js maneja una solicitud de archivo:

  1. Envía la tarea al sistema de archivos de la computadora.
  2. Listo para manejar la próxima solicitud.
  3. Cuando el sistema de archivos se ha abierto y leído el archivo, el servidor devuelve el contenido al cliente.

Node.js elimina la espera y simplemente continúa con la siguiente solicitud.

Node.js ejecuta programación asíncrona de un solo subproceso, sin bloqueo, que es muy eficiente en la memoria.

¿Qué puede hacer Node.js?

  • Node.js puede generar contenido de página dinámico
  • Node.js puede crear, abrir, leer, escribir, eliminar y cerrar archivos en el servidor
  • Node.js puede recopilar datos de formularios
  • Node.js puede agregar, eliminar, modificar datos en su base de datos

¿Qué es un archivo Node.js?

  • Los archivos Node.js contienen tareas que se ejecutarán en ciertos eventos
  • Un evento típico es alguien que intenta acceder a un puerto en el servidor
  • Los archivos Node.js deben iniciarse en el servidor antes de tener algun efecto
  • Los archivos Node.js tienen la extensión ".js"

Empezando con Node.js

Descarga Node.js

El sitio web oficial de Node.js tiene instrucciones de instalación para Node.js: https://nodejs.org

Una vez que haya descargado e instalado Node.js en su computadora, intentemos mostrar "Hola Mundo" en el navegador web.

Crea un archivo Node.js llamado "miprimer.js" y luego agrega el siguiente código:

var http = require('http');

        http.createServer(function (req, res) {
        res.writeHead(200, {'Content-Type': 'text/html'});
        res.end('¡Hola Mundo!');
        }).listen(8080);

Guarde el archivo en su computadora: C:\Users\Tu Nombre\miprimer.js

El código le dice a la computadora que escriba "¡Hola mundo!" si alguien (por ejemplo, un navegador web) intenta acceder a su computadora en el puerto 8080.

Por ahora, no es necesario que comprenda el código. Se explicará más adelante.

Línea de Comandos

Los archivos Node.js deben iniciarse en el programa "Interfaz de línea de comandos" de su computadora.

La forma de abrir la interfaz de línea de comandos en su computadora depende del sistema operativo. Para los usuarios de Windows, presione el botón de inicio y busque "Símbolo del sistema", o simplemente escriba "cmd" en el campo de búsqueda.

Navegue hasta la carpeta que contiene el archivo "miprimer.js", la ventana de la interfaz de línea de comandos debería verse así:

C:\Users\Tu Nombre>_

Iniciar el Archivo Node.js

Node.js debe iniciar el archivo que acabas de crear antes de que pueda llevarse a cabo cualquier acción.

Inicie su de línea de comandos, escriba node miprimer.js y presione enter:

C:\Users\Tu Nombre>node miprimer.js

¡Ahora tu computadora funciona como servidor!

Si alguien intenta acceder a su computadora en el puerto 8080, obtendrá un "¡Hola mundo!" mensaje a cambio!

Inicie su navegador de Internet y escriba la dirección: http://localhost:8080

Módulos de Node.js

Los módulos son lo mismo que las bibliotecas de JavaScript.

Un conjunto de funciones que deseas incluir en tu aplicación.

Módulos Incorporados

Node.js tiene un conjunto de módulos integrados que puedes usar sin ninguna instalación adicional.

Incluir módulos

Para incluir un módulo, use la función require() con el nombre del módulo:

var http = require('http');

Ahora su aplicación tiene acceso al módulo HTTP y puedes crear un servidor:

http.createServer(function (req, res) {
  res.writeHead(200, {'Content-Type': 'text/html'});
  res.end('¡Hola Mundo!');
}).listen(8080);

Crea tus propios módulos

Puedes crear tus propios módulos e incluirlos fácilmente en tus aplicaciones.

El siguiente ejemplo crea un módulo que devuelve un objeto de fecha y hora:

Ejemplo:

exports.fechaYHora = function () {
  return Date();
};

Utiliza la palabra clave exports para que las propiedades y los métodos estén disponibles fuera del archivo del módulo.

Guarda el código anterior en un archivo llamado "miprimermodulo.js"

Incluye tu módulo

Ahora puede incluir y usar el módulo en cualquiera de sus archivos Node.js.

Ejemplo:

var http = require('http');
var dt = require('./myfirstmodule');

http.createServer(function (req, res) {
  res.writeHead(200, {'Content-Type': 'text/html'});
  res.write("La fecha y la hora actual es: " + dt.fechaYHora());
  res.end();
}).listen(8080);

Resultado:

La fecha y la hora actual es: Fri Dec 31 2021 11:32:33 GMT-0500 (Colombia Standard Time)

Observa que usamos ./ para ubicar el módulo, eso significa que el módulo está ubicado en la misma carpeta que el archivo Node.js.

Guarde el código anterior en un archivo llamado "modulo_demo.js" e inicie el archivo:

C:\Users\Tu Nombre>node modulo_demo.js

Si has seguido los mismos pasos en tu computadora, verás el mismo resultado que en el ejemplo: http://localhost:8080

Módulo HTTP de Node.js

Node.js tiene un módulo integrado llamado HTTP, que permite a Node.js transferir datos a través del Protocolo de transferencia de hipertexto (HTTP).

Para incluir el módulo HTTP, use el método require():

var http = require('http');

Node.js como servidor web

El módulo HTTP puede crear un servidor HTTP que escucha los puertos del servidor y da una respuesta al cliente.

Utiliza el método createServer() para crear un servidor HTTP:

var http = require('http');

//create a server object:
http.createServer(function (req, res) {
  res.write('¡Hola Mundo!'); //escribe una respuesta al cliente
  res.end(); //fin de la respuesta
}).listen(8080); //el objeto del servidor escucha en el puerto 8080

La función pasada al método http.createServer() se ejecutará cuando alguien intente acceder a la computadora en el puerto 8080.

Guarde el código anterior en un archivo llamado "demo_http.js" e inicie el archivo:

C:\Users\Tu Nombre>node demo_http.js

Si ha seguido los mismos pasos en su computadora, verá el mismo resultado que en el ejemplo: http://localhost:8080

Agregar un encabezado HTTP

Si se supone que la respuesta del servidor HTTP se muestra como HTML, debes incluir un encabezado HTTP con el tipo de contenido correcto:

Ejemplo:

var http = require('http');
http.createServer(function (req, res) {
  res.writeHead(200, {'Content-Type': 'text/html'});
  res.write('¡Hola Mundo!');
  res.end();
}).listen(8080);

Resultado:

¡Hola Mundo!

El primer argumento del método res.writeHead() es el código de estado, 200 significa que todo está bien, el segundo argumento es un objeto que contiene los encabezados de respuesta.

Leer la cadena de consulta

La función pasada a http.createServer() tiene un argumento req que representa la solicitud del cliente, como objeto (objeto http.IncomingMessage).

Este objeto tiene una propiedad llamada "url" que contiene la parte de la URL que viene después del nombre de dominio:

var http = require('http');
http.createServer(function (req, res) {
  res.writeHead(200, {'Content-Type': 'text/html'});
  res.write(req.url);
  res.end();
}).listen(8080);

Guarda el código anterior en un archivo llamado "demo_http_url.js" e inicie el archivo:

C:\Users\Tu Nombre>node demo_http_url.js

Si has seguido los mismos pasos en su computadora, deberías ver dos resultados diferentes al abrir estas dos direcciones:

http://localhost:8080/verano

Producirá este resultado:

/verano

http://localhost:8080/invierno

Producirá este resultado:

/invierno

Dividir la cadena de consulta

Hay módulos integrados para dividir fácilmente la cadena de consulta en partes legibles, como el módulo URL.

Ejemplo:

var http = require('http');
var url = require('url');

http.createServer(function (req, res) {
  res.writeHead(200, {'Content-Type': 'text/html'});
  var q = url.parse(req.url, true).query;
  var txt = q.year + " " + q.month;
  res.end(txt);
}).listen(8080);

Guarde el código anterior en un archivo llamado "demo_cadenaconsulta.js" e inicie el archivo:

Inicia demo_cadenaconsulta.js:

C:\Users\Tu Nombre>node demo_cadenaconsulta.js

La dirección:

http://localhost:8080/?year=2017&month=julio

Producirá este resultado:

2017 July

Módulo File System

Node.js como un servidor de archivos

El módulo del sistema de archivos Node.js te permite trabajar con el sistema de archivos en su computadora.

Para incluir el módulo del sistema de archivos, utiliza el método require():

var fs = require('fs');

Uso común del módulo sistema de archivos:

  • Leer archivos
  • Crear archivos
  • Actualizar archivos
  • Borrar archivos
  • Cambiar el nombre de los archivos

Leer archivos

El método fs.readFile() se utiliza para leer archivos en su computadora.

Supongamos que tenemos el siguiente archivo HTML (ubicado en la misma carpeta que Node.js):

archivodemo1.html

<html>
<body>
<h1>Mi encabezado</h1>
<p>Mi párrafo</p>
</body>
</html>

Crea un archivo Node.js que lea el archivo HTML y devuelva el contenido:

Ejemplo:

var http = require('http');
var fs = require('fs');
http.createServer(function (req, res) {
  fs.readFile('archivodemo1.html', function(err, data) {
    res.writeHead(200, {'Content-Type': 'text/html'});
    res.write(data);
    return res.end();
  });
}).listen(8080);

Resultado:

Mi encabezado

Mi párrafo.

Guarde el código anterior en un archivo llamado "demo_readfile.js" e inicie el archivo:

Inicia demo_readfile.js:

C:\Users\Tu Nombre>node demo_readfile.js

Si ha seguido los mismos pasos en su computadora, verá el mismo resultado que en el ejemplo: http://localhost:8080

Crear archivos

El módulo sistema de archivos tiene métodos para crear nuevos archivos:

  • fs.appendFile()
  • fs.open()
  • fs.writeFile()

El método fs.appendFile() agrega contenido especificado a un archivo. Si el archivo no existe, se creará el archivo:

Ejemplo:

var fs = require('fs');

fs.appendFile('minuevoarchivo1.txt', '¡Hola Contenido!', function (err) {
  if (err) throw err;
  console.log('¡Archivo guardado!');
});
C:\Users\Tu Nombre>node demo_fs_append.js
¡Archivo guardado!

El método fs.open() toma una "bandera" como segundo argumento, si la bandera es "w" para "escribir", el archivo especificado se abre para escribir. Si el archivo no existe, se crea un archivo vacío:

Ejemplo:

Cree un archivo nuevo y vacío usando el método open():

var fs = require('fs');

fs.open('minuevoarchivo2.txt', 'w', function (err, file) {
  if (err) throw err;
  console.log('¡Archivo guardado!');
});

Resultado:

C:\Users\Tu Nombre>node demo_fs_open.js
¡Archivo guardado!

El método fs.writeFile() reemplaza el archivo y el contenido especificados si existe. Si el archivo no existe, se creará un nuevo archivo con el contenido especificado:

Ejemplo:

Cree un nuevo archivo usando el método writeFile():

var fs = require('fs');

fs.writeFile('minuevoarchivo3.txt', '¡Hola contenido!', function (err) {
  if (err) throw err;
  console.log('¡Archivo guardado!');
});

Actualizar archivos

El módulo sistema de archivos tiene métodos para actualizar archivos:

  • fs.appendFile()
  • fs.writeFile()

El método fs.appendFile() agrega el contenido especificado al final del archivo especificado:

Ejemplo:

Agregue "Este es mi texto". al final del archivo "minuevoarchivo1.txt":

var fs = require('fs');

fs.appendFile('minuevoarchivo1.txt', ' Este es mi texto.', function (err) {
  if (err) throw err;
  console.log('¡Archivo actualizado!');
});

El método fs.writeFile() reemplaza el archivo y el contenido especificados:

Ejemplo:

Reemplace el contenido del archivo "minuevoarchivo3.txt":

var fs = require('fs');

fs.writeFile('minuevoarchivo3.txt', 'Este es mi texto', function (err) {
  if (err) throw err;
  console.log('¡Remplazado!');
});

Resultado:

C:\Users\Tu Nombre>node demo_fs_replace.js
¡Remplazado!

Borrar archivos

Para eliminar un archivo con el sistema de archivos, usa el método fs.unlink() .

El método fs.unlink() elimina el archivo especificado:

Ejemplo:

Eliminemos el archivo "minuevoarchivo2.txt":

var fs = require('fs');

fs.unlink('minuevoarchivo2.txt', function (err) {
  if (err) throw err;
  console.log('¡Archivo Eliminado!');
});
C:\Users\Tu Nombre>node demo_fs_unlink.js
¡Archivo eliminado!

Cambiar el nombre de los archivos

Para cambiar el nombre de un archivo con el sistema de archivos, utiliza el método fs.rename().

El método fs.rename() cambia el nombre del archivo especificado:

Ejemplo:

Cambie el nombre de "minuevoarchivo1.txt" a "miarchivorenombrado.txt":

var fs = require('fs');

fs.rename('minuevoarchivo1.txt', 'miarchivorenombrado.txt', function (err) {
  if (err) throw err;
  console.log('¡Archivo renombrado!');
});

Resultado:

C:\Users\Tu Nombre>node demo_fs_rename.js
¡Archivo renombrado!

Subir archivos

También puede usar Node.js para cargar archivos a su computadora.

Módulo URL

El módulo de URL divide una dirección web en partes legibles.

Para incluir el módulo de URL, use el método require():

var url = require('url');

Analiza una dirección con el método url.parse() y devuelve un objeto URL con cada parte de la dirección como propiedades:

Ejemplo:

Divida una dirección web en partes legibles:

var url = require('url');
var adr = 'http://localhost:8080/default.htm?year=2017&month=february';
var q = url.parse(adr, true);

console.log(q.host); //devuelve 'localhost:8080'
console.log(q.pathname); //devuelve '/default.htm'
console.log(q.search); //devuelve '?year=2017&month=february'

var qdata = q.query; //devuelve un objeto: { year: 2017, month: 'february' }
console.log(qdata.month); //devuelve 'february'
C:\Users\Tu Nombre>node demo_url.js
localhost:8080
/default
?year=2017&month=february
february

Servidor de archivos de Node.js

Ahora sabemos cómo analizar la cadena de consulta, y en el capítulo anterior aprendimos cómo hacer que Node.js se comporte como un servidor de archivos. Combinemos los dos, y sirvamos el archivo solicitado por el cliente.

Crea dos archivos html y guárdelos en la misma carpeta que sus archivos node.js.

verano.html

<!DOCTYPE html>
<html>
<body>
<h1>Verano</h1>
<p>¡Yo amo el sol!</p>
</body>
</html>

invierno.html

<!DOCTYPE html>
<html>
<body>
<h1>Invierno</h1>
<p>¡Amo la nieve!</p>
</body>
</html>

Crea un archivo Node.js que abra el archivo solicitado y devuelva el contenido al cliente. Si algo sale mal, lanza un error 404:

demo_fileserver.js:

var http = require('http');
var url = require('url');
var fs = require('fs');

http.createServer(function (req, res) {
  var q = url.parse(req.url, true);
  var filename = "." + q.pathname;
  fs.readFile(filename, function(err, data) {
    if (err) {
      res.writeHead(404, {'Content-Type': 'text/html'});
      return res.end("404 No encontrado");
    } 
    res.writeHead(200, {'Content-Type': 'text/html'});
    res.write(data);
    return res.end();
  });
}).listen(8080);

Recuerda iniciar el archivo:

C:\Users\Tu Nombre>node demo_fileserver.js

http://localhost:8080/verano.html

Producirá este resultado:

Verano

¡Yo amo el sol!

http://localhost:8080/nieve.html

Producirá este resultado:

Invierno

¡Amo la nieve!

NPM

¿Qué es NPM?

NPM es un administrador de paquetes para paquetes Node.js, o módulos si lo desea.

www.npmjs.com aloja miles de paquetes gratuitos para descargar y usar.

El programa NPM se instala en su computadora cuando instala Node.js

Nota: ¡NPM ya está listo para ejecutarse en su computadora!

¿Qué es un paquete?

Un paquete en Node.js contiene todos los archivos que necesita para un módulo.

Los módulos son las bibliotecas de JavaScript que puedes incluir en tu proyecto.

Descargar un paquete

Descargar un paquete es muy sencillo.

Abra la de línea de comandos y dígale a NPM que descargue el paquete que desea.

Quiero descargar un paquete llamado "upper-case":

C:\Users\Tu Nombre>npm install upper-case

¡Ahora ha descargado e instalado su primer paquete!

NPM crea una carpeta llamada "node_modules", donde se colocará el paquete. Todos los paquetes que instale en el futuro se colocarán en esta carpeta.

Mi proyecto ahora tiene una estructura de carpetas como esta:

C:\Users\Tu Nombre\node_modules\upper-case

Usando un paquete

Una vez que el paquete está instalado, está listo para usar.

Incluya el paquete "upper-case" de la misma manera que incluye cualquier otro módulo:

var uc = require('upper-case');

Crea un archivo Node.js que convertirá el resultado "¡Hola mundo!" en letras mayúsculas:

Ejemplo:

var http = require('http');
var uc = require('upper-case');
http.createServer(function (req, res) {
  res.writeHead(200, {'Content-Type': 'text/html'});
  res.write(uc.upperCase("¡Hola Mundo!"));
  res.end();
}).listen(8080);

Resultado:

¡HOLA MUNDO!

Guarde el código anterior en un archivo llamado "demo_uppercase.js" e inicie el archivo:

C:\Users\Tu Nombre>node demo_uppercase.js

Si has seguido los mismos pasos en tu computadora, verás el mismo resultado que en el ejemplo: http://localhost:8080
