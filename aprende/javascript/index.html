Tutorial de JavaScript

Introducción

JavaScript es el lenguaje de programación más popular del mundo, es el lenguaje de programación de la web y es muy fácil de aprender.

¿Por qué estudiar JavaScript?

JavaScript es uno de los 3 lenguajes que todos los desarrolladores web deben aprender:

  1. HTML para definir el contenido de las páginas web.
  2. CSS para especificar el diseño de las páginas web.
  3. JavaScript para programar el comportamiento de las páginas web.

Este tutorial cubre todas las versiones de JavaScript:

  • El JavaScript original ES1 ES2 ES3 (1997-1999)
  • La primera revisión principal ES5 (2009)
  • La segunda revisión ES6 (2015)
  • Las adiciones anuales (2016, 2017, 2018)

Velocidad de aprendizaje

En este tutorial, la velocidad de aprendizaje es de tu elección.

Todo depende de ti.

Si tienes dificultades, tómate un descanso o vuelve a leer el material.

Siempre asegúrate de comprender todos los ejemplos.

La única forma de convertirte en un programar inteligente es: Practicar.

Preguntas frecuentes

  • ¿Cómo obtengo JavaScript?
  • ¿Dónde puedo descargar JavaScript?
  • ¿JavaScript es gratuito?

Respuestas

No es necesario que obtengas o descargues JavaScript.

JavaScript ya se está ejecutando en tu navegador, en tu computadora, en tu tableta o teléfono inteligente.

JavaScript es de uso gratuito para todos.

JavaScript puede cambiar el contenido HTML

Uno de los muchos métodos HTML de JavaScript es getElementById().

El siguiente ejemplo "busca" un elemento HTML (con el id="demo") y cambia el contenido del elemento (innerHTML) a "Hola JavaScript":

Ejemplo:

<!DOCTYPE html>
<html>
<body>

<h2>¿Qué puede hacer JavaScript?</h2>

<p id="demo">JavaScrit puede cambiar el contenido HTML.</p>

<button type="button" onclick='document.getElementById("demo").innerHTML = "¡Hola JavaScript!"'>¡Haz clic en mí!</button>

</body>
</html>

Resultado:

¿Qué puede hacer JavaScript?

JavaScript puede cambiar el contenido HTML.

JavaScript acepta comillas simples y comillas dobles:

document.getElementById('demo').innerHTML = 'Hola JavaScript';

JavaScript puede cambiar los valores de los atributos HTML

En este ejemplo, JavaScript cambie el valor del atributo src de una etiqueta <img>.

Ejemplo:

<!DOCTYPE html>
<html>
<body>

<h2>¿Qué puede hacer JavaScript?</h2>

<p>JavaScript puede cambiar los valores de los atributos HTML.</p>

<p>En este caso JavaScript cambia el valor del atributo src (source) de una imagen.</p>

<button onclick="document.getElementById('miImagen').src='pic_bulbon.gif'">Encender la luz</button>

<img id="miImagen" src="pic_bulboff.gif" style="width:100px">

<button onclick="document.getElementById('myImage').src='pic_bulboff.gif'">Apagar la luz</button>

</body>
</html>

Resultado:

¿Qué puede hacer JavaScript?

JavaScript puede cambiar los valores de los atributos HTML.

En este caso JavaScript cambia el valor del atributo src (source) de una imagen.

JavaScript puede cambiar los estilos HTML (CSS)

Cambiar el estilo de un elemento HTML es una variante de cambiar un atributo HTML:

Ejemplo:

<!DOCTYPE html>
<html>
<body>

<h2>¿Qué puede hacer JavaScript?</h2>

<p id="demo">JavaScript puede cambiar el estilo de un elemento HTML.</p>

<button type="button" onclick="document.getElementById('demo').style.fontSize='35px'">Haz clic en mí</button>

</body>
</html>

Resultado:

¿Qué puede hacer JavaScript?

JavaScript puede cambiar el estilo de un elemento HTML.

JavaScript puede ocultar elementos HTML

Se pueden ocultar elementos HTML cambiando el estilo de la propiedad de CSS display.

Ejemplo:

<!DOCTYPE html>
<html>
<body>

<h2>¿Qué puede hacer JavaScript?</h2>

<p id="demo">JavaScript puede ocultar elementos HTML.</p>

<button type="button" onclick="document.getElementById('demo').style.display='none'">¡Haz clic en mí!</button>

</body>
</html>

Resultado:

¿Qué puede hacer JavaScript?

JavaScript puede ocultar elementos HTML.

JavaScript puede mostrar elementos HTML

También se puede mostrar elementos HTML ocultos cambiando la propiedad display.

Ejemplo:

<!DOCTYPE html>
<html>
<body>

<h2>¿Qué puede hacer JavaScript?</h2>

<p>JavaScript puede mostrar elementos HTML ocultos.</p>

<p id="demo" style="display:none">¡Hola JavaScript!</p>

<button type="button" onclick="document.getElementById('demo').style.display='block'">¡Haz clic en mí!</button>

</body>
</html>

Resultado:

¿Qué puede hacer JavaScript?

JavaScript puede mostrar elementos HTML ocultos.

¡Hola JavaScript!

¿Sabías qué?...

JavaScript y Java son lenguajes completamente diferentes, tanto en concepto como en diseño.

JavaScript fue inventado por Brendan Eich en 1995 y se convirtió en un estándar ECMA en 1997.

ECMA-262 es el nombre oficial del estándar. EcmaScript es el nombre oficial del lenguaje.
