Contador de Carácteres


El contador de caracteres cuenta el número de caracteres ingresados ​​en el campo de texto. Ayuda a mostrar la notificación sobre la cantidad de caracteres escritos en el cuadro de texto. Si desea mostrar el recuento de caracteres ingresados ​​o el recuento de caracteres restantes en el área de texto, el contador de caracteres es la mejor opción para eso.

JAVASCRIPT
let cadena = document.getElementById("cadena");

let imprimir = (mensaje) => {
    let salida = document.getElementById("salida");
    salida.innerHTML = "El largo de caracteres es: " + "" + mensaje + "";
}

cadena.addEventListener("keyup", e =>{
    if(e){
        e.preventDefault();
        imprimir(document.getElementById('cadena').value.length);
    }
}, false);
