Reproductor MP3


El reproductor MP3 es un proyecto simple desarrollado con JavaScript, CSS y HTML. Este es un proyecto interesante para escuchar nuestra propia colección de canciones. El usuario puede reproducir diferentes tipos de música y disfrutar del sonido de su música. El usuario puede hacer clic en el botón de reproducción hacia adelante y hacia atrás para cambiar las canciones.
